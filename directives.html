Basic Directives

  • {{ $variable }}: Echo a variable.
  • {!! $variable !!}: Echo a variable without escaping.
  • {{-- This is a comment --}}: Blade comment.

    • Control Structures

  • @if (condition): Start an if statement.
  • @elseif (condition): Start an else-if statement.
  • @else: Start an else statement.
  • @endif: End an if statement.
  • @unless (condition): The inverse of @if.
  • @endunless: End the unless statement.
  • @isset (variable): Check if a variable is set.
  • @endisset: End the isset statement.
  • @empty (variable): Check if a variable is empty.
  • @endempty: End the empty statement.
  • @auth: Check if the user is authenticated.
  • @endauth: End the auth statement.
  • @guest: Check if the user is a guest (not authenticated).
  • @endguest: End the guest statement.
  • @switch (variable): Start a switch statement.
  • @case (value): Start a case block within switch.
  • @break: Break out of the switch statement.
  • @default: Default case within switch.
  • @endswitch: End the switch statement.

    • Loops

  • @for ($i = 0; $i < 10; $i++): Start a for loop.
  • @endfor: End the for loop.
  • @foreach ($array as $item): Start a foreach loop.
  • @endforeach: End the foreach loop.
  • @forelse ($array as $item): Start a foreach loop with an empty check.
  • @empty: Executes if the array is empty.
  • @endforelse: End the forelse loop.
  • @while (condition): Start a while loop.
  • @endwhile: End the while loop.

    • Loops Control

  • @continue: Skip to the next iteration of the loop.
  • @break: Exit the loop.

    • Include Statements

  • @include('view.name'): Include a view.
  • @includeIf('view.name'): Include a view if it exists.
  • @includeWhen(condition, 'view.name'): Include a view if a condition is true.
  • @includeUnless(condition, 'view.name'): Include a view if a condition is false.
  • @includeFirst(['view.name', 'view.fallback']): Include the first view that exists.

    • Components

  • @component('component.name'): Start defining a component.
  • @endcomponent: End the component definition.
  • @slot('slotName'): Define a slot within a component.
  • @endslot: End the slot definition.
  • : Include a Blade component.

    • Forms

  • @csrf: Generate a CSRF token field.
  • @method('PUT'): Spoof the HTTP verb.
    • Custom Directives
  • @directive('name'): Define a custom directive.
  • @enddirective: End the custom directive.

    • Section and Yield

  • @section('sectionName'): Start defining a section.
  • @endsection: End the section definition.
  • @yield('sectionName'): Yield a section's content.
  • @show: End the section and yield its content immediately.
  • @append: Append content to the section.

    • Extending Layouts

  • @extends('layout.name'): Extend a Blade layout.
  • @parent: Insert content from the parent layout.

    • Stacks

  • @push('stackName'): Push content onto a stack.
  • @endpush: End the stack push.
  • @prepend('stackName'): Prepend content onto a stack.
  • @endprepend: End the stack prepend.
  • @stack('stackName'): Render the stack.

    • Other Directives

  • @inject('name', 'ClassName'): Inject a class instance into the view.
  • @json($array): Convert an array to JSON.
