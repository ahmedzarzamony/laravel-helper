Core Components

  • Routing: Defines how your application responds to different URL paths.
  • Middleware: Filters HTTP requests entering your application.
  • Controllers: Handles incoming requests and returns responses.
  • Requests: Manages incoming HTTP requests.
  • Responses: Manages outgoing HTTP responses.

    • Database and Eloquent

  • Eloquent ORM: Provides an active record implementation for interacting with the database.
  • Query Builder: Allows for constructing SQL queries programmatically.
  • Migrations: Manages database schema changes.
  • Seeders: Populates the database with test data.
  • Factories: Creates fake data for testing.

    • Views and Blade

  • Blade Templating: A powerful templating engine for rendering views.
  • View Composer: Attaches data to views globally.

    • Authentication and Authorization

  • Authentication: Manages user login and registration.
  • Authorization: Manages access control and permissions.

    • Sessions and Cookies

  • Session: Manages user session data.
  • Cookies: Manages HTTP cookies.

    • Caching

  • Cache: Provides a unified API for various caching backends like Redis and Memcached.

    • Events and Broadcasting

  • Events: Provides a simple observer pattern implementation.
  • Broadcasting: Broadcasts server-side events to the client-side using WebSockets.

    • Queues

  • Queue: Manages background jobs.

    • File Storage

  • Filesystem: Provides an abstraction layer for file storage, supporting local, Amazon S3, and others.

    • HTTP Client

  • HTTP Client: Provides a fluent interface for making HTTP requests.

    • Validation

  • Validation: Provides a powerful validation system.

    • Localization

  • Localization: Manages translation and localization of application content.

    • Notifications

  • Notifications: Sends notifications across various channels like email, SMS, and Slack.

    • Mail

  • Mail: Provides a clean, simple API for sending emails.

    • Artisan Console

  • Artisan: Command-line interface included with Laravel.

    • Testing

  • Testing: Provides tools for unit testing, feature testing, and browser testing.

    • API Resources

  • API Resources: Transforms your models and collections into JSON.

    • Broadcasting

  • Echo: A JavaScript library that makes it easy to subscribe to channels and listen for events broadcast by Laravel.

    • Logging

  • Logging: Provides a unified API for various logging backends.

    • Task Scheduling

  • Task Scheduling: Provides an expressive, fluent interface for defining command schedules.

    • Notifications

  • Notification Channels: Integrates with different notification channels (email, SMS, Slack, etc.).

    • Package Development

  • Package Development: Provides tools and guidance for creating reusable packages.
